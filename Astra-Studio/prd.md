# Astra Studio — 多模态 AI 工作台
## 产品需求文档 (PRD)

---

## 1. 产品概述

### 1.1 产品名称
**Astra Studio** — 多模态 AI 创作工作台

### 1.2 产品定位
面向创作者、设计师和内容生产者的专业级 AI 辅助创作平台，支持对话、图像生成、视频生成、语音识别、代码辅助等多模态能力。

### 1.3 目标用户
- 内容创作者（播客、自媒体、编辑）
- 视觉设计师
- 开发者
- 需要多模态 AI 辅助的专业人士

---

## 2. 核心功能模块

### 2.1 侧边栏导航 (Sidebar)

#### 2.1.1 品牌区域
- Logo 标识（品牌首字母 + 品牌名称）
- 工作空间切换器（支持多工作空间管理）

#### 2.1.2 创作工具集
| 功能 | 图标 | 状态 |
|------|------|------|
| 对话 | message-square | 默认激活 |
| 图像生成 | image | 可用 |
| 视频生成 | video | BETA |
| 语音识别 | audio-lines | 可用 |
| 代码助手 | code-2 | 可用 |
| 翻译润色 | languages | 可用 |

#### 2.1.3 资料库
- 最近记录（显示数量）
- 收藏（显示数量）
- 项目（显示数量）

#### 2.1.4 用户信息
- 用户头像与名称
- 算力余额显示
- 设置入口

---

### 2.2 主对话区域 (Main)

#### 2.2.1 顶部导航栏
- 面包屑导航（创作 / 对话标题）
- 标题可编辑
- 模型选择器（显示当前模型 + 状态指示）
- 操作按钮：分享、历史、更多

#### 2.2.2 对话展示区
- 用户消息（左侧头像 + 时间戳 + 内容）
- AI 回复（品牌头像 + 思考时间 + 富文本内容）
- 支持 Markdown 格式渲染
- 支持附件卡片展示

#### 2.2.3 附件卡片
- 图像预览（1:1 比例）
- 元信息展示（模型、分辨率、用时）
- 操作按钮：重新生成、放大、下载

#### 2.2.4 输入区 (Composer)
- 多行文本输入
- 工具栏：
  - 附件上传
  - 图像模式切换
  - 语音输入
  - 联网搜索
- Token 计数显示
- 发送按钮
- 快捷键提示（↵ 发送、⇧↵ 换行、/ 调用工具）

---

### 2.3 工作室面板 (Studio)

#### 2.3.1 模型配置
- 当前模型显示
- 模型标签（推理、长上下文、多模态）
- 模型切换下拉

#### 2.3.2 参数调节
| 参数 | 默认值 | 调节方式 |
|------|--------|----------|
| 温度 (Temperature) | 0.72 | 滑块 |
| 最大输出 (Max Tokens) | 4,096 | 滑块 |
| Top-p | 0.95 | 滑块 |

#### 2.3.3 系统提示词
- 可编辑的系统提示词区域
- 支持富文本格式

#### 2.3.4 会话产物展示
- 网格布局展示本次会话生成的内容
- 支持类型：图像、视频、语音、代码
- 显示元信息（尺寸、时长、用时）

---

### 2.4 设置面板 (Tweaks)

#### 2.4.1 主题切换
- 深色模式（默认）
- 浅色模式

#### 2.4.2 强调色选择
- 琥珀色（默认）
- 青苔色
- 陶土色

#### 2.4.3 文字密度
- 宽松
- 默认
- 紧凑

---

## 3. 设计规范

### 3.1 色彩系统

#### 3.1.1 深色模式（默认）
| 变量 | 用途 | OKLCH 值 |
|------|------|----------|
| --bg | 主背景 | 17% 0.008 70 |
| --bg-2 | 次级背景 | 20% 0.008 70 |
| --bg-3 | 三级背景 | 23% 0.009 70 |
| --border | 边框 | 28% 0.01 70 |
| --text | 主文字 | 96% 0.005 85 |
| --text-2 | 次级文字 | 72% 0.01 80 |
| --accent | 强调色 | 82% 0.13 65 |

#### 3.1.2 状态色
| 状态 | 颜色 |
|------|------|
| 成功 | oklch(76% 0.13 155) |
| 警告 | oklch(82% 0.14 90) |
| 危险 | oklch(68% 0.18 25) |

### 3.2 字体系统
| 字体族 | 用途 |
|--------|------|
| Instrument Serif | 品牌标识、标题、强调文本 |
| Geist | 正文、UI 元素 |
| Geist Mono | 代码、数值、等宽显示 |

### 3.3 布局规范
| 区域 | 尺寸 |
|------|------|
| 侧边栏宽度 | 248px |
| 工作室面板宽度 | 336px |
| 顶部导航高度 | 56px |

### 3.4 响应式断点
| 断点 | 行为 |
|------|------|
| ≤1180px | 隐藏工作室面板 |
| ≤760px | 隐藏侧边栏，单列布局 |

## 4. 交互规范

### 4.1 导航交互
- 当前激活项左侧显示强调色指示条
- Hover 状态背景色变化
- 徽章（Badge）显示功能状态

### 4.2 对话交互
- 用户消息左对齐
- AI 回复带思考时间显示
- 附件卡片支持悬停操作

### 4.3 输入交互
- 聚焦时边框变为强调色
- 工具芯片支持激活状态
- Token 计数实时更新

### 4.4 参数调节
- 滑块支持拖拽调节
- 数值实时显示
- 滑块填充色使用强调色

## 5. 技术实现

### 5.1 前端技术栈
- 纯 HTML/CSS/JavaScript
- 无框架依赖
- Lucide 图标库
- Google Fonts（Geist、Instrument Serif）

### 5.2 CSS 特性
- CSS 变量（Custom Properties）
- OKLCH 色彩空间
- Grid 布局
- Flexbox 布局
- CSS 动画与过渡

### 5.3 浏览器兼容
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 支持 CSS OKLCH 的浏览器
